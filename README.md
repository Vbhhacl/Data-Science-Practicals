# 📊 Data Science Practicals – SPPU

This repository contains all my Data Science practical assignments completed as part of the **Savitribai Phule Pune University (SPPU)** curriculum.  
The goal is to maintain a structured collection of practical exercises that demonstrate the application of Python, data analysis, and machine learning concepts on real-world datasets.

---

## 📘 Practicals Overview

### 🔹 Practical 1 – Introduction to Data Science
- Basics of Python programming for data analysis
- Loading datasets and inspecting data
- Exploring data types and basic statistics
- Understanding rows, columns, and indexing in pandas

### 🔹 Practical 2 – Data Wrangling and Preprocessing
- Data loading from CSV or web sources (e.g., Kaggle)
- Checking and handling missing values
- Data cleaning, type conversion, and normalization
- Converting categorical variables to numerical
- Summarizing dataset statistics and variable types
- Example dataset: Iris dataset
- Demonstrates `.head()`, `.tail()`, `.describe()`, `.isnull()`, `.fillna()`, `.sort_values()`, `.loc[]`, `.iloc[]`, and transpose operations

### 🔹 Practical 4 – Regression Analysis and Predictive Modeling
- Working with structured datasets like Boston Housing
- Splitting dataset into features and target
- Train-test split for model evaluation
- Implementing Linear Regression
- Measuring model performance using metrics
